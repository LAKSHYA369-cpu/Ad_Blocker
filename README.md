# 🛡️ AuraShield Ad Blocker

**AuraShield** is a lightweight, efficient, and powerful ad blocker extension compatible with both **Google Chrome** and **Microsoft Edge**. It blocks annoying ads, trackers, and pop-ups to give you a clean and lightning-fast browsing experience.

---

## 🚀 How to Install and Use AuraShield

Since AuraShield is currently shared as a developer package, you can easily install it manually using **Developer Mode** in your browser. Follow these steps:

### Step 1: Download and Extract the Files
1. Download the `aurashield.zip` file from this repository.
2. Right-click the downloaded `.zip` file and select **Extract All...** to unzip it.
3. Inside, you will find the main folder named **`aurashield`** (this folder contains files like `manifest.json`). Keep this folder in a safe place on your PC where you won't accidentally delete it.

---

### Step 2: Add AuraShield to Your Browser

#### 🌐 For Google Chrome:
1. Open Google Chrome and type `chrome://extensions/` in the address bar, then press **Enter**.
2. In the top-right corner of the Extensions page, turn **ON** the toggle switch for **Developer mode**.
3. Click the **Load unpacked** button in the top-left corner.
4. Browse your computer, select the extracted **`aurashield`** folder, and click **Select Folder**.
5. AuraShield is now installed! Click the **Puzzle piece icon** next to your address bar to find AuraShield and **Pin** it to your toolbar.

#### 🟦 For Microsoft Edge:
1. Open Microsoft Edge and type `edge://extensions/` in the address bar, then press **Enter**.
2. In the left sidebar, turn **ON** the toggle switch for **Developer mode**.
3. Click the **Load unpacked** button at the top of the page.
4. Browse your computer, select the extracted **`aurashield`** folder, and click **Select Folder**.
5. Click the **Puzzle piece icon** on your toolbar and click the **Show in toolbar (eye icon)** next to AuraShield to keep it visible.

---

## 🧪 How to Test If It's Working

To make sure AuraShield is working perfectly and blocking ads successfully, follow these steps:

1. Go to the official test website: [https://adblock.turtlecute.org/](https://adblock.turtlecute.org/)
2. The website will automatically scan to see if ad elements are being blocked.
3. Look at your score! If AuraShield is running correctly, it will show that ads are blocked successfully.

---

## ⏸️ How to Temporarily Disable AuraShield

If you ever need to turn off the ad blocker:

1. Open your browser's extension manager page (`chrome://extensions/` or `edge://extensions/`).
2. Find **AuraShield** in your list of installed extensions.
3. Simply **toggle the blue switch to OFF**. 
4. To turn it back on, just click the toggle switch again.
